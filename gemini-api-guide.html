Scribe API Setup Guide

Follow these steps to get a free API key and activate your Scribe.

1 Visit Google AI Studio

Open Google AI Studio (aistudio.google.com) in a new tab.

2 Sign In

Sign in using your standard Google or Gmail account.

3 Generate API Key

  • Click on "Get API key" in the top left sidebar menu.
  • Click the blue button: "Create API key".
  • Choose "Create API key in new project" (if prompted).
  • Copy the generated key to your clipboard.

4 Activate the Assistant

Return to Scribe, paste your key in the sidebar, and click "Activate".

Privacy & Security:
Your API key is stored locally in your browser and is only sent directly to Google's official API endpoints. We do not store or see your key.
